DJI Protobuf Tags

Tags found in protobuf-format DJI meta djmd and dbgi timed metadata. Only a few tags are currently known, but unknown djmd tags may be extracted by setting the Unknown option to 1 (or 2 to also extract unknown dbgi debug tags). Tag ID's are composed of the corresponding .proto file name combined with the hierarchical protobuf field numbers.

Sony LensType Values

Decimal values have been added to differentiate lenses which would otherwise have the same LensType, and are used by the Composite LensID tag when attempting to identify the specific lens model. "New" or "II" appear in brackets if the original version of the lens has the same LensType. Special logic is employed to identify the attached lens when a Metabones Canon EF adapter is used.
